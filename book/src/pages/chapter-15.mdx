# Chapter 15: Block vs Point Technique

When using row-column scanning, how should groups be highlighted?

Block technique trades speed for clarity; point technique reduces visual clutter but requires more visual precision. Try both with real users.

## Block technique

- Entire row lights up
- Easy to see
- Best for beginners and low vision

## Point technique

- Only row indicator highlights
- Less visual clutter
- Best for dense grids and experienced users

<div class="info-box">
  Best practice: start with block technique for beginners, then experiment with point technique.
</div>
