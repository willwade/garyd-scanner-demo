# Chapter 16: Continuous Scanning

Continuous scanning moves focus smoothly rather than stepping item-by-item.

- Gliding cursor
- Crosshair
- Compass (8-direction)

<div class="info-box">
  Continuous scanning is timing-sensitive. If users miss targets, slow scan rate or add acceptance time.
</div>
