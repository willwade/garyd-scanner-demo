# Chapter 19: Error Recovery

Errors are expensive for switch users. Provide recovery tools that are always available.

## Recovery methods

- **Empty items:** Selectable items that reset the scan without output
- **Cancel scan:** Dedicated cancel action (button, long-hold, or special item)
- **Undo/redo:** Reverse last character or operation
- **Back navigation:** Return to previous group/page

<div class="warning-box">
  Without recovery, a single mistake can cost 30+ seconds. This is the most common reason users abandon scanning systems.
</div>
