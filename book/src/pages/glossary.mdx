# Glossary

<h2 id="glossary-term-acceptance-time">Acceptance Time</h2>
<p>Time a switch must be held before selection is confirmed.</p>

<h2 id="glossary-term-auto-scan">Auto-Scan</h2>
<p>System automatically advances highlight at a set rate.</p>

<h2 id="glossary-term-continuous-scan">Continuous Scanning</h2>
<p>Focus moves smoothly rather than stepping item-by-item.</p>

<h2 id="glossary-term-critical-overscan">Critical Overscan</h2>
<p>Fast forward scan with slow-back precision selection.</p>

<h2 id="glossary-term-directed-scan">Directed Scanning</h2>
<p>User steers highlight using directional switches.</p>

<h2 id="glossary-term-dwell-time">Dwell Time</h2>
<p>Auto-select when highlight stays on item for a threshold duration.</p>

<h2 id="glossary-term-elimination-scan">Elimination Scanning</h2>
<p>Repeatedly split the set until the target remains.</p>

<h2 id="glossary-term-group-scan">Group Scan</h2>
<p>Select a group first, then scan within that group.</p>

<h2 id="glossary-term-input-bandwidth">Input Bandwidth</h2>
<p>Amount of information a user can transmit per unit time.</p>

<h2 id="glossary-term-row-column">Row-Column Scanning</h2>
<p>Scan rows, then items in the selected row.</p>

<h2 id="glossary-term-scan-rate">Scan Rate</h2>
<p>Time each item stays highlighted.</p>

<h2 id="glossary-term-selection-set">Selection Set</h2>
<p>All items available for selection at a given time.</p>

<h2 id="glossary-term-step-scan">Step Scan</h2>
<p>User controls movement and selection.</p>
