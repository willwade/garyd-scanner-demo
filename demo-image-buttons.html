‚ú® Feature: Switch scanner now supports realistic image-based buttons instead of plain text! Image buttons are enabled by default and show different colors for each switch in elimination mode.

1. Default Mode (Blue Buttons)

The default configuration shows blue image buttons for all actions.

Configuration:

<switch-scanner grid-size="16"></switch-scanner>

(Image buttons are enabled by default, using blue color)

2. Custom Color (Green)

You can choose any of the 4 built-in colors.

Configuration:

<switch-scanner grid-size="16" button-color="green"></switch-scanner>

3. Elimination Scanning (Multi-Color!)

In elimination mode, each switch gets a different color for easy identification!

Switch 1 (Top-Left)
Switch 2 (Top-Right)

Configuration:

<switch-scanner scan-pattern="elimination" elimination-switch-count="4"></switch-scanner>

(Each quadrant gets a different colored switch!)

4. Text Mode (Disabled)

You can disable image buttons to use traditional text buttons.

Configuration:

<switch-scanner grid-size="16" use-image-button="false"></switch-scanner>
